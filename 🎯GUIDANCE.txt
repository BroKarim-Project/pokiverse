=> Disini kita akan fetch data dari pokeApi sehingga nantinya kita bisa menampilkan nama dan gambar dari pokemon

1. fetch data
- Mengambil data dari API menggunkana fetch, ketika kita fetch data kita akan mendaapt respon HTTP dari server dan kita simpan di variable response
- The awaited response object holds the response from the server, but it's still in JSON format
      
2. convert response to js object
- kita konveris respons menjadi objek js, note sekalipun nama methjod ini .json() resultnya buaan JSON namun kiya terima inpu JSON dan konvert ke js objek

3. Set State for next & previous url
Atur state nextURL dan previousUrl dengan nilai yang diperoleh dari respons API

4. creates an array of promises for detailed Pokemon information based on URLs
- selanjutany kita membuat array baru bernaam resultPromis, dari mana tahu dia array? karena ada map function
- selanjtnya kita asumsikan data memilki property bernama results, kita akan melakukan iterasi dari setiap elemmen pokemon yang terdapat di data.results
- selanjutnya kita membuat callback function, di dalamnya kita melkaukan fetch lagi untuk mengambil url yang ada di pokemon
- seperti biaasa setelah kita fetch url itu kita harus konveris lagi kedalam objek js menggunakn method .json
      
5. Save the array in pokemonData
- promise bermkana janji, ibarat kata kita pesan nasi pada pelayan, secara tidak langsung pelayan itu akan berjanji akan mengantar jika ada atau menolak jika menu taka ada
- kode ini berguna untuk menunggu detail pokemon di fetch
 setelah semua berhasil di dapat )fetch) itu akan di simpan di pokemonData
- sevenarnya kita dapat langsung menggunakna data di resultPromises tanpa melakukan promise.all, namun untuk lebih memastikan agar semua data benar2 di fetch, lbh mudah menagangi kesalah karena ada banyak pokemon dan kit tak tahu mana iterasi yang bermasalash
